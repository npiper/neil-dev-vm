# Mule Developer VM

```
As an API/Integration Developer
I need an SOE that has all required software pre-installed
so that I can quickly be productive 
And there is a consistent toolset for our Devs
```
- [x] Java Oracle JDK 1.8
- [x] Chrome
- [x] Postman
- [x] Mulesoft IDE
- [x] SoapUI
- [x] Docker
- [ ] ActiveMQ
- [x] HermesJMS
- [ ] MiniKube (Kubernetes)
- [ ] MiniShift
- [ ] Ansible
- [ ] Puppet
